Users

Login

This endpoint logs the user in for their session.

HTTP Request

POST http://devwars.tv/v1/user/login

The above command returns JSON structured like this:

{
   "id":0,
   "username":"sdfj",
   "email":"terencebezman@gmail.com",
   "ranking":{
      "id":0,
      "points":0.0
   },
   "role":"USER"
}

Query Parameters

Parameter Default Description
username NONE The username of the user signing in
password NONE The password of the user signing in
Remember that this request also returns a cookie with their the user’s session token!

Sign Up

This endpoint creates a user account.

HTTP Request

POST http://devwars.tv/v1/user/create

The above command returns either a message about an email being sent or the error message:

Query Parameters

Parameter Default Description
username NONE The username of the user signing up
email NONE The email of the user igning up
password NONE The password of the user signing up
Remember that this request will send an email to the user’s email if there were no conflicts!

Get User

This retrieves a user with a given id.

HTTP Request

GET http://devwars.tv/v1/user/{id}

The above command returns JSON structured like this:

{
   "id":0,
   "username":"sdfj",
   "email":"terencebezman@gmail.com",
   "ranking":{
      "id":0,
      "points":0.0
   },
   "role":"USER"
}
Remember that you must be signed in as at least a moderator to perform this action

Remove a user

This retrieves a user with a given id.

HTTP Request

GET http://devwars.tv/v1/user/{id}/delete

The above command returns JSON structured like this for the user just deleted:

{

   "username":"sdfj",
   "email":"terencebezman@gmail.com",
   "ranking":{
      "id":0,
      "points":0.0
   },
   "role":"USER"
}
Remember that you must be signed in as at least a moderator to perform this action

Add points

This adds points to the specified user.

HTTP Request

GET http://devwars.tv/v1/user/{id}/addpoints

The above command returns JSON structured like this:

{
   "id":0,
   "username":"sdfj",
   "email":"terencebezman@gmail.com",
   "ranking":{
      "id":0,
      "points":3.0
   },
   "role":"USER"
}
Parameter Default Description
points NONE The amount of points to add to the user
Don’t be afraid to add negative points to subtract from the user

Update Game

This creates a DevWars game with a default placeholders.

HTTP Request

GET http://devwars.tv/v1/game/{id}/update

The above command returns JSON structured like this:

{
   "teams":{
      "red":{
         "embedLink":"Default embed link",
         "players":{

         },
         "name":"red",
         "id":1,
         "game_id":0,
         "status":"Waiting"
      },
      "blue":{
         "embedLink":"Default embed link",
         "players":{

         },
         "name":"blue",
         "id":2,
         "game_id":0,
         "status":"Waiting"
      }
   },
   "name":"Default Name",
   "active":false,
   "id":0,
   "timestamp":1424638562000
}

A very important note to remember is that when you update something, make sure you change only the primary key / the index of the item you wish to change So to add someone new on the red team with the language HTML do the following

{
   "teams":{
      "red":{
         "embedLink":"Default embed link",
         "players":{
            "HTML":{
               "id": 1 //This would be the user's id in the database
            }
         },
         "name":"red",
         "id":1,
         "game_id":0,
         "status":"Waiting"
      },
      "blue":{
         "embedLink":"Default embed link",
         "players":{

         },
         "name":"blue",
         "id":2,
         "game_id":0,
         "status":"Waiting"
      }
   },
   "name":"Default Name",
   "active":false,
   "id":0,
   "timestamp":1424638562000
}

Query Parameters

Parameter Default Description
id NONE The id of the game you will update
json NONE An entire JSON object for a game
